== Description ==
 
Easily display all the WordPress standard information required for any plugins you have developed on their own separate pages. Create plugin tags for easy re-use on any plugin page. Easily update the plugin pages you have created with custom input fields for each plugin attribute. The plugin display page template meshes well with any theme but conforms to the standard WordPress style for displaying plugins. You won't forget what information you need to include about your plugin, the named inputs will remind you! 
 
== Installation ==
 
Download the plugin from our website or by searching Plugin Display Page in the Plugin Browser.

1. Install and activate the plugin
2. Click 'Plugin Pages' in the admin menu
3. Fill in the inputs and click publish
4. Click View Post to check out your brand new plugin display page!
